# Jharia Land Use/Land Cover (LULC) Change Detection Analysis

## Overview
This project analyses Land Use/Land Cover (LULC) changes in the Jharia Coalfield, Jharkhand, using QGIS and satellite imagery. The study evaluates temporal changes in mining areas, vegetation, water bodies, and built-up land to understand the environmental and land-use impacts of mining activities.

---

## Objectives
- Quantify changes in the mining area over time.
- Assess variations in vegetation cover.
- Analyse changes in water bodies.
- Evaluate the expansion of built-up areas.
- Visualise land-use transitions using GIS-based thematic maps.

---

## Methodology
1. Acquired multi-temporal satellite imagery.
2. Preprocessed and clipped imagery to the study area.
3. Generated LULC classification maps.
4. Calculated area statistics for each land-use class.
5. Compared LULC changes across different years.
6. Interpreted the spatial and environmental impacts through thematic maps and statistical analysis.

## Key Findings
- Identified changes in **mining area** associated with mining expansion.
- Evaluated changes in **vegetation cover** over the study period.
- Analyzed variations in **water bodies** using multi-temporal satellite imagery.
- Assessed the growth of **built-up areas** due to urban and industrial development.
- Produced GIS-based thematic maps and area statistics to support spatial analysis.
